## Learn Git Branching

链接：[learngitbranching.js.org]<https://learngitbranching.js.org/>

## 笔记

`git branch <name>` 新建<name>分支

`git checkout <name>` 切换到分支<name>

`git checkout -b <name>` 创建并切换到<name>分支
